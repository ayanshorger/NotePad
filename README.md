# NotePad
A simple and functional Notepad application built using Python's Tkinter library.  
This project is designed to replicate basic features of a text editor like Windows Notepad with additional enhancements.

🚀 Features

📄 New File, Open File, Save & Save As
✂️ Cut, Copy, Paste, Delete
🔍 Find Text (Highlight Search)
⏱️ Insert Date & Time
🔄 Undo & Redo Support
🔤 Select All Text
📏 Word Wrap Toggle
🔎 Zoom In / Zoom Out (Menu + Ctrl + Mouse Scroll)
🖱️ Smooth Mouse-based Zoom Functionality
📌 User-friendly GUI using Tkinter
